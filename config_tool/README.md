# STM32L051 Modbus 配置上位机

基于项目寄存器地址表的 Web UI 配置工具，浏览器打开即用。

## 快速开始

```bash
# 1. 安装依赖
pip install pyserial

# 2. 启动
python app.py

# 3. 打开浏览器
# http://localhost:8080
```

指定端口：
```bash
python app.py 9090
```

## 功能

- **串口连接**: 自动扫描可用串口，支持 9600~115200 波特率
- **系统配置**: 读写 UART 波特率、校验位、Modbus 地址、上报格式等
- **从机配置**: 5 个从机 Tab 页，独立配置地址、轮询周期、数据点
- **数据点配置**: 每从机 8 个数据点，支持 U16/I16/U32/I32/Float + 4 种字节序
- **设备命名**: 支持中文名称（GBK/UTF-8），最大 20 字符
- **JSON 导入/导出**: 配置备份与批量部署
- **通信日志**: 实时显示 Modbus 通信详情

## 使用流程

1. 将 STM32L051 的 **PA15 接高电平**（进入配置模式/从站模式）
2. 连接 RS485 转换器到电脑
3. 启动本工具，选择串口和波特率
4. 点击「连接」→「读取全部配置」
5. 修改参数后点击「写入全部配置」
6. 将 PA15 接低电平（回到轮询上报模式）

## 寄存器地址表

| 地址范围 | 说明 |
|----------|------|
| 0x0000~0x000A | 系统配置（波特率、地址、格式等） |
| 0x0100+N×0x0200 | 从机 N 基本配置 + 名称 |
| 0x0100+N×0x0200+0x0010+M×0x0010 | 从机 N 数据点 M 配置 + 名称 |

## 注意事项

- 设备必须处于**配置模式**（PA15=HIGH）才能读写配置
- 写入配置后会自动保存到 EEPROM（延迟至主站 IDLE 时执行）
- 批量写入使用 0x10 功能码，减少 EEPROM 写入次数
- 名称字段支持 ASCII/GBK/UTF-8 编码
